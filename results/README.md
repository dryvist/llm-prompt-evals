## Key finding

> **The value of a good system prompt is inversely proportional to how strong the runner is.** On top-tier runners (Opus 4.8, Fable 5) even the bare `base` prompt scores near-perfect — the model compensates for a weak prompt. On cheaper/weaker runners (DeepSeek Flash) the disciplined prompts pull real weight. Evaluate on the model you will actually deploy.
